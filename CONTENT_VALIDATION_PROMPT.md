# Content Validation Prompt - QA Agent

## Validation Checklist

### Critical Requirements (Must Pass)
- [ ] 2,000-5,000 words
- [ ] 5+ credible citations with URLs
- [ ] Proper H1>H2>H3 hierarchy
- [ ] TechnicalTerm components used correctly
- [ ] Functional code examples
- [ ] FAQ section (5-8 questions)
- [ ] References section numbered

### Technical Accuracy
- [ ] All claims source-backed
- [ ] Performance metrics include conditions
- [ ] Current info (<12 months old)
- [ ] Security follows best practices
- [ ] Code tested and functional

### Content Quality
- [ ] Professional tone maintained
- [ ] Actionable implementation steps
- [ ] Real-world examples included
- [ ] Primary keyword in title/first paragraph
- [ ] 3-5 internal links present

### Guardrails Compliance
- [ ] No vendor promotion
- [ ] No copied content
- [ ] No unsubstantiated claims
- [ ] No outdated information

## Rejection Criteria
Reject if: <2,000 words, <5 citations, factual errors, promotional content, non-functional code, plagiarism
