## Overview

This is a minimal project that demonstrates the new optional schema. It has:

- No stats (stars, forks, etc.)
- No external links (GitHub, demo, docs)
- Only the required fields: topics, description, and content

## Features

- Clean and simple
- Focuses on content
- No external dependencies

## Usage

This project shows that you can create portfolio entries without needing GitHub stats or external links. Perfect for:

- Private projects under NDA
- Conceptual work
- Internal tools
- Case studies without code

The Stats and Links cards will not be displayed on the detail page since no data is provided.
