## Overview

spectral-ui is a fully accessible React component library built with design tokens, dark/light theming, and zero runtime CSS dependencies. Every component meets WCAG 2.1 AA standards.

## Quick Start

```bash
npm install spectral-ui
```

```tsx
import { Button, Card } from 'spectral-ui';

function App() {
  return (
    <Card>
      <Button variant="primary">Click me</Button>
    </Card>
  );
}
```

## Features

- 50+ production-ready components
- Full WCAG 2.1 AA compliance
- Design token system
- Dark/light theme support
- Zero runtime CSS (CSS-in-JS at build time)
- TypeScript-first API
- Comprehensive Storybook documentation

## Components

Button, Card, Modal, Dropdown, Tabs, Accordion, Toast, Tooltip, Input, Select, Checkbox, Radio, Switch, Slider, Progress, Badge, Avatar, and more.
